# Note

When we declare a function, we can omit the function name to create an anonymous function.

```js
// anonymous function
const add1 = function (a, b) {
  const sum = a + b
  return sum
}

// anonymous function with arrow syntax
const add2 = (a, b) => a + b

console.log(add1(3, 5))
console.log(add2(5, 3))
```
